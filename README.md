# 视频内容识别工具

这个Python脚本使用智谱AI的GLM-4.6V-Flash模型来识别视频中的语音内容，并将其转换为文本格式。

## 功能

- 读取视频文件并将其转换为base64编码
- 使用智谱AI的视觉语言模型识别视频中人物所说的内容
- 将识别结果按照特定格式输出：
  - 不区分说话人
  - 内容写在一行
  - 添加适当的标点符号
  - 使用逗号分隔分句和停顿，语义完整时使用句号
  - 仅返回复述内容，不包含额外说明
  - 如果无法识别内容，返回"无法识别"

## 安装依赖

在使用此脚本前，请确保安装以下Python包：

```bash
pip install zhipuai
```

## 使用方法

1. 确保您有一个有效的智谱AI API密钥，[智谱官网](https://bigmodel.cn/)
2. 修改脚本中的API密钥为您自己的密钥
3. 更改脚本末尾的`video_path`变量为您要处理的视频文件路径
4. 运行脚本：

```bash
python process.py
```

## 注意事项

- 确保视频文件与脚本在同一目录下，或提供完整的文件路径
- API密钥需要保密，不要将其提交到公共代码库
- 视频文件大小可能受API限制，如遇到问题，请尝试使用较小的视频文件
- 此脚本需要网络连接以访问智谱AI的API服务
